Diikuti tombol inline keyboard di Telegram:
[ 📋 Funneling ]  [ 📅 Activity ]
[    📊 Performansi    ]
Atau bisa diakses via perintah:
/funneling · /activity · /performansi

Catatan teknis untuk tombol:
Di Telegram Bot API, tombol ini namanya Inline Keyboard — tinggal tambahkan reply_markup saat kirim pesan /start:
json{
  "inline_keyboard": [
    [
      {"text": "📋 Funneling", "callback_data": "/funneling"},
      {"text": "📅 Activity",  "callback_data": "/activity"}
    ],
    [
      {"text": "📊 Performansi", "callback_data": "/performansi"}
    ]
  ]
}
Setiap kali tombol diklik, bot langsung balas dengan data terkait AM tersebut
